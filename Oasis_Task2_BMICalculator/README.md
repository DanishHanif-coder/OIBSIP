# Oasis Infobyte - Python Programming Internship

## Task 2: BMI Calculator

A console-based application to calculate Body Mass Index (BMI) based on user-provided weight (kg) and height (m).

### Features:
* Accurate BMI calculation using standard formulas.
* Automatic categorization (Underweight, Normal, Overweight, Obese) with tips.
* Secure input validations to ignore negative or string values.

### How to Run:
Run the script using Python:
```bash
python main.py
